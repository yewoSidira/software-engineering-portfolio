Featured Project

CampusConnect

A university engagement platform designed to centralize announcements, events, clubs, and communication while demonstrating scalable backend architecture.

Problem

Universities often rely on fragmented communication channels including WhatsApp groups, notice boards, Facebook pages, and email. This leads to missed announcements and inconsistent information.

Solution

CampusConnect provides a centralized platform where university administrators, student organizations, and clubs can publish verified announcements, manage events, and communicate efficiently with students.

Engineering Decisions

  • REST API architecture using Flask
  • PostgreSQL for relational data integrity
  • Redis for caching and future background jobs
  • Dockerized development environment
  • Role-based authentication and permissions
